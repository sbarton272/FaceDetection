This is the baseline algorith for the 18-794 F14 contest.
It correlates the zero-mean average face from the training data
with the frame at various sizes and uses non-maximal suppression and
thresholding to find faces.

Negative Examples from: http://www.robots.ox.ac.uk/~vgg/data/data-cats.html
- used Google "things" dataset and Caltech Background dataset
